# Penyelesaian
Kita Diberi File Dengan Nama "ret" Dan kita disuruh untuk mengetahui Password dari file ini. <br>
Karena file "ret" Adalah mempunyai heading "ELF" Yang berarti file executable.
![image](https://github.com/NeloRedgrave/WriteUp_CTF/assets/140806028/1a0d6b5e-1457-475f-b848-227c3e7637d9)
Sebelum kita menajalankan perintah untuk menjalankan file "ret" kita akan melakukan <br>
chmod < mengganti perizinan sebuah file > pada file "ret" agar bisa dijalankan 
![image](https://github.com/NeloRedgrave/WriteUp_CTF/assets/140806028/d210f7eb-933a-44cf-82f1-2c56a7da1857)
Dengan begini kita bisa menjalankan file "ret"
![image](https://github.com/NeloRedgrave/WriteUp_CTF/assets/140806028/57abf059-4d87-43ef-897e-3192fd17fae0)
Karena kita tidak tahu apapun mengenai passwordnya , maka kita akan melakukan Reverse Pada file ini. <br>
Dengan cara kita akan mengubah code dari file ini.
